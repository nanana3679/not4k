# not4k — 차트 편집기

이 문서는 차트 편집기가 다루는 차트의 데이터 모델을 정의한다.

---

## 차트 메타데이터

하나의 차트 파일에 포함되는 곡 및 채보 정보.

| 필드       | 설명                                |
| ---------- | ----------------------------------- |
| 곡명       | 곡의 제목                                           |
| 아티스트명 | 곡의 아티스트                                       |
| 난이도명   | 채보의 난이도 라벨 (예: EASY/NORMAL/HARD)           |
| 난이도     | 차트 레벨 (Lv.) 수치                                |
| 이미지     | 곡 선택 화면 등에 표시되는 자켓 이미지              |
| 음원파일   | 곡의 오디오 파일                                    |
| 오프셋     | 음원 재생 시작으로부터 0박까지의 시간 차이 (ms 단위) |

---

## 엔티티

차트를 구성하는 배치 가능한 요소. 모든 엔티티는 **위치**를 가진다.

### 위치

| 속성   | 설명                                                                 |
| ------ | -------------------------------------------------------------------- |
| 레인   | L1~L4. 노트에는 필수, 박자 마커와 BPM 마커에는 없음 (레인에 속하지 않는 전역 엔티티) |
| 박자수 | 곡 시작(0박)으로부터의 경과 박자. **분수**로 표기한다                 |

**박자수의 분수 표기**: 정수 박자는 `0`, `1`, `4`처럼 표기하고, 박자 내 세분 위치는 분수로 표기한다. 예를 들어 8비트의 두 번째 틱은 `1/2`, 16비트의 세 번째 틱은 `3/4`. 이를 통해 비트 분할에 독립적인 위치 표현이 가능하다.

---

### 노트

레인 위에 배치되는 입력 대상. 위치의 레인이 필수이다.

| 노트 종류              | 설명                                                                |
| ---------------------- | ------------------------------------------------------------------- |
| 싱글 노트              | 레인에 바인딩된 키 중 하나를 누르는 노트                            |
| 더블 노트              | 레인에 바인딩된 서로 다른 키 2개를 동시에 누르는 노트               |
| 트릴 노트              | 직전 입력과 다른 키를 눌러야 하는 노트                              |
| 싱글 롱노트 바디 시작  | 싱글 롱노트 바디 구간의 시작점. 대응하는 바디 끝을 참조한다         |
| 싱글 롱노트 바디 끝    | 싱글 롱노트 바디 구간의 끝점. 대응하는 바디 시작을 참조한다         |
| 더블 롱노트 바디 시작  | 더블 롱노트 바디 구간의 시작점. 대응하는 바디 끝을 참조한다         |
| 더블 롱노트 바디 끝    | 더블 롱노트 바디 구간의 끝점. 대응하는 바디 시작을 참조한다         |
| 트릴 롱노트 바디 시작  | 트릴 롱노트 바디 구간의 시작점. 대응하는 바디 끝을 참조한다         |
| 트릴 롱노트 바디 끝    | 트릴 롱노트 바디 구간의 끝점. 대응하는 바디 시작을 참조한다         |

**바디 시작과 끝의 상호 참조**: 롱노트 바디 시작과 바디 끝은 반드시 쌍으로 존재하며, 서로를 참조한다. 바디 시작 없는 바디 끝, 또는 바디 끝 없는 바디 시작은 유효하지 않다.

---

### 박자 마커

한 마디에 몇 박자가 들어가는지를 정의하는 전역 엔티티. 레인에 속하지 않는다.

- **0박에 반드시 존재**한다. 곡의 시작 시점에서 박자가 정의되어야 한다.
- **마디가 시작하는 박자에만 배치**할 수 있다.
- 다시 등장하면 **해당 시점 이후로 박자가 변경**된다. 다음 박자 마커가 나올 때까지 유효하다.

예: 0박에 `4/4`, 32박에 `3/4` → 0~31박은 4박자 마디, 32박 이후는 3박자 마디.

---

### BPM 마커

템포를 정의하는 전역 엔티티. 레인에 속하지 않는다.

- **0박에 반드시 존재**한다. 곡의 시작 시점에서 BPM이 정의되어야 한다.
- 다시 등장하면 **해당 시점 이후로 BPM이 변경**된다. 다음 BPM 마커가 나올 때까지 유효하다.

예: 0박에 `BPM 150`, 64박에 `BPM 180` → 0~63박은 150 BPM, 64박 이후는 180 BPM.
