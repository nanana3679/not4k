# not4k — 난이도 설계

이 문서는 not4k의 난이도가 어떤 구조로 상승하는지, 각 등급에서 유저가 무엇을 학습하는지를 정의한다.

---

## 설계 원칙

### 1. IIDX 12레벨 붕괴를 반복하지 않는다

IIDX의 공식 최고 레벨인 12레벨에는 입문급(지력 F)부터 인간 한계급(지력 S+)까지 섞여 있다. 게임 내 UI는 이 격차를 설명해주지 않으며, 유저는 반드시 커뮤니티의 비공식 난이도표를 참조해야 한다. not4k는 충분히 넓은 레벨 범위와 난이도 등급 체계로 이 문제를 방지한다.

### 2. DJMAX 5B의 가변 손배치 학습 단절을 반복하지 않는다

DJMAX 5B에서 가변 손배치는 고난이도의 레이저 패턴에서만 강제된다. 저난이도 유저는 가변 손배치를 경험할 기회가 없어서, 고난이도에 도달했을 때 급격한 난이도 벽을 만난다. not4k는 트릴 노트를 통해 저난이도에서부터 가변 손배치를 점진적으로 학습시킨다.

### 3. 성장 경로가 하나로 연결된다

8키 바인딩을 기본으로 설정하되 입문에서는 4키를 주로 활용하며, 10키까지 확장되지만 레인 구조는 항상 동일하다. DJMAX의 4B→5B→6B처럼 키 수가 바뀔 때마다 처음부터 다시 적응할 필요가 없다. 이전 등급에서 학습한 능력이 다음 등급의 기반이 된다.

### 4. 모든 노트 타입은 Lv.1부터 등장한다

난이도에 따라 변하는 것은 노트 타입의 등장 여부가 아니라, **빈도와 복합도**이다. 가장 쉬운 곡에서도 트릴·더블·홀드 중 탭이 극소량 등장하여, 유저는 첫 곡부터 "이 게임은 기존 4키와 다르다"는 것을 체감한다.

---

## 난이도 체계

- **Lv. (차트 레벨)**: 채보 하나에 부여되는 난이도 수치.
- **PLv. (피스 레벨)**: 개별 피스가 본격 활용되는 최소 레벨.
- **난이도 등급**: 레벨 구간을 묶은 분류(입문/초급/중급/상급/최상급).

> **용어 규칙**: 채보 단위는 **Lv.**, 피스 단위는 **PLv.**, 구간 분류는 **난이도 등급**으로 통일한다.

---

## 난이도 등급 정의

### 입문 (Lv.1~3)

| 항목               | 내용                                               |
| ------------------ | -------------------------------------------------- |
| **유저 키 바인딩** | 4키 주로 활용 (중지-검지), 드물게 엄지             |
| **핵심 경험**      | 4레인 인식, 기본 타이밍, "이 게임은 다르다"를 인지 |
| **노트 구성**      | 싱글 90%↑ / 트릴·더블 극소량 등장                  |

8키 바인딩을 기본으로 설정하되, 입문 단계에서는 중지와 검지 4키를 주로 활용한다. 트릴 노트와 더블 노트는 내측 레인(2, 3번)에서만 극소량 등장하여, 드물게 엄지를 사용하게 된다. "이 게임은 기존 4키와 다르다"는 인식을 심어주면서도, 기존 4K 유저가 거의 그대로 플레이할 수 있는 수준을 유지한다.

---

### 초급 (Lv.4~6)

| 항목               | 내용                                       |
| ------------------ | ------------------------------------------ |
| **유저 키 바인딩** | 엄지 비중 상승 (6키)                       |
| **핵심 경험**      | 트릴 구간이 명확해지고, 더블이 흐름에 삽입 |
| **선행 학습**      | 4레인 인식, 기본 타이밍                    |

엄지의 비중이 점점 높아지는 단계다. 트릴 구간이 입문보다 길어지고 빈번해지며, 더블 노트가 리듬의 강조점에 삽입된다. 릴리즈 교차(PLv.4)가 본격 활용되기 시작한다. 내측 레인(2, 3번)에서 검지+엄지를 적극 활용하게 되지만, 외측 레인(1, 4번)은 아직 주로 1키만 사용한다.

---

### 중급 (Lv.7~10)

| 항목               | 내용                                 |
| ------------------ | ------------------------------------ |
| **유저 키 바인딩** | 8키 입문 (약지 추가)                 |
| **핵심 경험**      | 트릴 앵커 본격화, 홀드 중 탭 등장  |
| **선행 학습**      | 트릴 처리, 더블 노트 처리, 엄지 활용 |

외측 레인(1, 4번)에 약지가 추가되어 8키를 본격적으로 사용하기 시작하는 단계다. 트릴 앵커가 본격화되어, 한 손이 트릴을 처리하는 동안 반대 손이 복잡한 배치를 담당하는 구조가 등장한다. 홀드 중 탭(PLv.7)이 등장하여 한 레인 내에서 다층 입력을 요구한다.

기존 4K 게임에서 연타 구간에서는 불가능했던 조합이 등장하며, not4k의 핵심 차별점이 드러나는 난이도대다.

---

### 상급 (Lv.11~13)

| 항목               | 내용                                   |
| ------------------ | -------------------------------------- |
| **유저 키 바인딩** | 8키 활성화                             |
| **핵심 경험**      | 모든 노트 타입이 높은 밀도로 복합 결합 |
| **선행 학습**      | 8키 기본 활용, 홀드 중 탭 처리       |

모든 피스가 복합적으로 결합된다. 가변 분할 전환이 등장하여 좌우 손의 담당 레인이 실시간으로 바뀌고, 높은 인지/물리 부하가 동시에 요구된다.

---

### 최상급 (Lv.14~)

| 항목               | 내용                                    |
| ------------------ | --------------------------------------- |
| **유저 키 바인딩** | 8키 완전 활용, 10키 가능                |
| **핵심 경험**      | 기존 4키 게임에 존재하지 않는 차트 공간 |
| **선행 학습**      | 모든 피스의 복합 처리                   |

한 레인에서 롱노트를 유지하면서 동시에 싱글 노트를 처리하는 배치가 고밀도로 등장한다. 일부 레인에 3키를 바인딩(총 10키)하여 더블 홀드 + 싱글 노트 처리 등을 수행할 수 있다. 건너가기가 등장하여 손의 물리적 교차까지 요구된다.

이 단계에서 not4k는 기존의 어떤 4키 리듬게임에도 없었던 차트 공간에 진입한다.

---

## 등급 간 전이 설계

각 등급의 경계에서 유저가 느끼는 "벽"을 최소화하기 위한 원칙:

### 1. 온보딩 차트

각 등급의 최초 곡들은 새로운 피스를 매우 낮은 밀도로 도입한다. 예를 들어 초급의 첫 곡은 릴리즈 교차가 곡 전체에서 4~8회만 등장하고, 나머지는 입문 수준의 배치로 구성된다.

### 2. 역방향 동기 부여

입문에서 연타로 막히는 곡을 경험한 유저에게, "키를 하나 더 활용하고 같은 곡을 다시 해보세요"라는 제안을 할 수 있는 시스템이 필요하다. 같은 곡이 키 활용에 따라 다른 체감 난이도를 가진다는 것을 직접 경험하게 하는 것이 가장 강력한 온보딩이다.

### 3. 난이도 표시

곡의 난이도 표기에 Lv.와 난이도 등급을 명시한다. IIDX에서 12레벨 내부의 격차를 게임이 설명해주지 않아 유저가 외부 자료에 의존해야 했던 문제를 방지한다.
