# not4k — 옵저버 모드

> **구현 상태: 미구현**

실제로 플레이하지 않고 차트를 확인하는 모드.

---

## 존재 이유

maimai나 Sound Voltex처럼, 고난이도 차트에서는 인지적 부하가 높다. 유저가 차트를 처음 보고 바로 플레이하면, 노트를 읽는 것과 손배치를 판단하는 것이 동시에 요구되어 실력과 무관하게 실패할 수 있다.

옵저버 모드는 **플레이 전 손배치 계획**을 가능하게 한다. 차트를 미리 확인하고, 어느 구간에서 어떤 손배치를 사용할지 사전에 결정할 수 있다. 이것은 not4k의 가변 손배치 시스템에서 특히 중요하다 — 레인당 다중키 바인딩 환경에서는 "어떤 키를 어떤 손가락으로 누를 것인가"라는 판단이 플레이의 핵심이며, 이 판단을 실시간이 아닌 사전에 수행할 수 있는 기회가 필요하다.

---

## 핵심 기능

### 1. 자유 스크롤

차트 내에서 자유롭게 스크롤하여 원하는 구간에 쉽게 도달할 수 있어야 한다.

- 차트 전체를 시간축으로 탐색할 수 있다.
- 스크롤 위치에 따라 현재 시점의 노트 배치가 표시된다.
- 빠른 이동(곡 시작/끝, 특정 시점 점프)을 지원한다.

### 2. 재생

차트를 음원과 함께 재생하여 실제 플레이 흐름을 확인할 수 있다.

- **재생/일시정지**: 현재 스크롤 위치에서 재생을 시작하고, 일시정지로 특정 시점에서 멈출 수 있다. 일시정지 중에도 자유 스크롤이 가능하다.
- **재생 속도 조절**: 감속 재생(0.25x~1.0x)으로 복잡한 배치를 정밀하게 확인한다.

### 3. 구간 반복 재생

스크롤 바 위에 **시작 마커**와 **끝 마커**를 배치하여 반복 재생 구간을 지정한다.

- **마커 없음**: 재생 시 곡 전체를 처음부터 끝까지 재생한다.
- **마커 있음**: 재생 시 마커 구간만 반복 재생한다. 끝 마커에 도달하면 시작 마커로 되돌아간다.
- **마커 조작**: 스크롤 바에서 드래그하여 시작/끝 위치를 조절한다. 마커 제거 시 전체 재생으로 복귀한다.

#### UI

```
┌─────────────────────────────────────────────┐
│  [◀ 시작] [▶ 재생/일시정지]  [속도: 0.5x ▼] │  ← 상단 컨트롤 바
├─────────────────────────────────────────────┤
│                                             │
│            (차트 노트 표시 영역)              │
│                                             │
├─────────────────────────────────────────────┤
│  A────[███████████]────B                    │  ← 스크롤 바 + 구간 마커
│  ↑시작마커              ↑끝마커               │    A~B 구간이 반복 재생 대상
└─────────────────────────────────────────────┘
```

- **상단 컨트롤 바**: 재생/일시정지 버튼, 곡 시작으로 이동, 재생 속도 선택
- **스크롤 바**: 곡 전체 타임라인. 현재 재생 위치를 표시하며, 드래그로 자유 이동
- **구간 마커**: 스크롤 바 위에 시작(A)·끝(B) 마커를 드래그로 배치. 마커가 있으면 구간이 하이라이트 표시되어 반복 범위를 시각적으로 확인할 수 있다
